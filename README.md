## 已注册的环境变量

| 名称                        | 说明                              |
| --------------------------- | --------------------------------- |
| process.env.CLI_HOME        | 用户主目录                        |
| process.env.CLI_HOME_PATH   | 用户主目录                        |
| process.env.CLI_TARGET_PATH | 是否指定本地调试文件路径          |
| process.env.LOG_LEVEL       | 日志的级别控制，一般用于控制debug |
